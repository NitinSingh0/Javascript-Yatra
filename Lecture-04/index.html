JavaScript has two main types of data:

  • Primitive
  • Object (also called Non-Primitive)

ðŸ“Œ Primitive Types (7 Total)

Primitive values are immutable and hold a single value:

  • Number: e.g. 42, 3.14
  • String: e.g. "Hello", 'World'
  • Boolean: true or false
  • Undefined: A variable declared but not assigned
  • Null: Intentionally set to "no value"
  • BigInt: Large integers, e.g. 12345678901234567890n
  • Symbol: Unique, immutable value, often used as object keys
let age = 25; // Number
let name = "Nitin"; // String
let isOnline = true; // Boolean
let score; // Undefined
let emptyValue = null; // Null
let bigNumber = 12345678901234567890n; // BigInt
let id = Symbol("userId"); // Symbol

Objects can hold multiple values and are used to store collections or more complex entities:

  • Object: Key-value pairs
  • Array: Indexed list of items
  • Function: Callable block of code
